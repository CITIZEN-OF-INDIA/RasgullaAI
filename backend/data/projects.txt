Project 1: GPT 5.O - Offline First Real-Time Chat Application

Description:
A scalable cross-platform chat system supporting Web, Desktop, and Android platforms.

Technologies:
React
TypeScript
Node.js
Socket.IO
MongoDB
Express.js
Electron.js

Key Features:
- Real-time messaging
- Presence tracking
- Delivery receipts
- Read receipts
- Offline-first synchronization
- IndexedDB storage
- Automatic conflict-safe synchronization
- JWT Authentication
- Desktop application packaging using Electron
